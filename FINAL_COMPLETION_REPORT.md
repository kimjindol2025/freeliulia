## 🏆 프로젝트 개요

**FreeLiulia**는 한글로 프로그래밍할 수 있는 **완전한 자체호스팅 컴파일러**를 갖춘 프로그래밍 언어입니다.

```
한글 소스 (.fl) → FreeLiulia 컴파일러 → C 코드 → gcc → 네이티브 바이너리
```

## 💡 기술 특징

### 언어 설계

```freeliulia
# 한글 키워드를 사용한 명확한 프로그래밍
함수 인사(이름: 문자열) -> 문자열
    변수 메시지 = "안녕하세요, " + 이름
    반환 메시지
끝

# 타입 추론
변수 점수 = 85        # 정수로 자동 추론
변수 가격 = 19.99     # 실수로 자동 추론
변수 활성 = 참        # 논리값으로 자동 추론

# 배열 및 함수형 프로그래밍
변수 점수들 = [85, 90, 78, 92, 88]
변수 합계 = 배열합계(점수들)
변수 평균 = 배열평균(점수들)
```

### 컴파일 파이프라인

```
입력 (FreeLiulia 코드)
    ↓
[1] Lexer (토크나이저)
    - 89개 토큰 타입 인식
    ↓
[2] Parser (파서)
    - AST 생성 (20+ 노드 타입)
    ↓
[3] Lowering (로워링)
    - Untyped IR 생성
    - 제어흐름 정규화 (goto 기반)
    ↓
[4] Type Inference (타입 추론)
    - Typed SSA IR 생성
    - 8가지 타입 시스템
    ↓
[5] Optimizer (최적화)
    - DCE
    - Inlining
    - Loop Unrolling
    ↓
[6] Codegen (코드생성)
    - C 코드 생성
    ↓
[7] gcc (C 컴파일)
    - 네이티브 바이너리
    ↓
출력 (실행 가능한 프로그램)
```

### 자체호스팅

```
FreeLiulia 소스 (compiler_v3.fl)
    ↓ [FreeLiulia 컴파일러로 컴파일]
C 코드 (compiler_v3.c)
    ↓ [gcc로 컴파일]
컴파일러 바이너리 (compiler_v3_bin1)
    ↓ [자신을 다시 컴파일]
컴파일러 바이너리 (compiler_v3_bin2)
    ↓ [또 다시 컴파일]
컴파일러 바이너리 (compiler_v3_bin3)
    ↓ [검증: Gen1.c == Gen2.c == Gen3.c ✅]
자체호스팅 성공! 🎉
```
