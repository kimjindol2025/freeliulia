## 📚 함수 카테고리

### 📝 문자열 함수 (10개)

#### 1. `길이(문자열) → 정수`
문자열의 길이를 반환합니다.

```freeliulia
변수 텍스트 = "안녕하세요"
변수 크기 = 길이(텍스트)  # 5
```

#### 2. `포함(문자열, 부분문자열) → 논리값`
문자열이 부분 문자열을 포함하는지 확인합니다.

```freeliulia
변수 있음 = 포함("hello world", "world")  # 참
```

#### 3. `시작(문자열, 접두어) → 논리값`
문자열이 특정 값으로 시작하는지 확인합니다.

```freeliulia
변수 시작함 = 시작("Python", "Py")  # 참
```

#### 4. `끝(문자열, 접미어) → 논리값`
문자열이 특정 값으로 끝나는지 확인합니다.

```freeliulia
변수 끝남 = 끝("hello.txt", ".txt")  # 참
```

#### 5. `찾기(문자열, 부분문자열) → 정수`
부분 문자열의 위치를 찾습니다 (없으면 -1).

```freeliulia
변수 위치 = 찾기("hello world", "world")  # 6
```

#### 6. `대문자(문자열) → 문자열`
문자열을 대문자로 변환합니다.

```freeliulia
변수 결과 = 대문자("hello")  # "HELLO"
```

#### 7. `소문자(문자열) → 문자열`
문자열을 소문자로 변환합니다.

```freeliulia
변수 결과 = 소문자("HELLO")  # "hello"
```

#### 8. `연결(문자열1, 문자열2) → 문자열`
두 문자열을 연결합니다.

```freeliulia
변수 결과 = 연결("안녕", "하세요")  # "안녕하세요"
```

#### 9. `자르기(문자열, 시작, 길이) → 문자열`
문자열의 부분을 추출합니다.

```freeliulia
변수 결과 = 자르기("hello world", 0, 5)  # "hello"
```

#### 10. `바꾸기(문자열, 기존, 새것) → 문자열`
문자열의 부분을 다른 것으로 바꿉니다.

```freeliulia
변수 결과 = 바꾸기("hello world", "world", "FreeLiulia")
```

### 🌟 고급 문자열 함수 (5개) [Phase 7 신규]

#### 11. `반복(문자열, 횟수) → 문자열`
문자열을 N회 반복합니다.

```freeliulia
변수 반복됨 = 반복("=", 5)  # "====="
변수 패턴 = 반복("*", 3)    # "***"
```

#### 12. `채우기(문자열, 길이) → 문자열`
문자열을 지정된 길이까지 왼쪽에 공백을 패딩합니다.

```freeliulia
변수 채워진 = 채우기("hello", 10)  # "     hello"
변수 정렬 = 채우기("42", 5)        # "   42"
```

#### 13. `나누기_앞(문자열, 구분자) → 문자열`
첫 번째 구분자까지의 앞부분만 반환합니다.

```freeliulia
변수 앞 = 나누기_앞("hello,world", ",")  # "hello"
```

#### 14. `나누기_뒷(문자열, 구분자) → 문자열`
첫 번째 구분자 이후의 뒷부분을 반환합니다.

```freeliulia
변수 뒷 = 나누기_뒷("hello,world", ",")  # "world"
```

### 📁 파일 I/O 함수 (8개) [Phase 7 신규]

#### 1. `파일열기(경로, 모드) → 파일핸들`
파일을 열고 핸들을 반환합니다.

```c
파일핸들 handle = 파일열기("data.txt", "r");
```

**모드**:
- `"r"`: 읽기
- `"w"`: 쓰기 (덮어쓰기)
- `"a"`: 추가 (append)

#### 2. `파일닫기(파일핸들) → 정수`
파일을 닫습니다. 성공시 0, 실패시 -1.

```c
파일닫기(handle);
```

#### 3. `줄읽기(파일핸들) → 문자열`
파일에서 한 줄을 읽습니다 (최대 1024자).

```c
char* line = 줄읽기(handle);
```

#### 4. `줄쓰기(파일핸들, 내용) → 정수`
파일에 한 줄을 씁니다.

```c
줄쓰기(handle, "Hello World");
```

#### 5. `파일존재(경로) → 논리값`
파일이 존재하는지 확인합니다.

```c
bool exists = 파일존재("data.txt");
```

#### 6. `파일삭제(경로) → 정수`
파일을 삭제합니다. 성공시 0, 실패시 -1.

```c
파일삭제("data.txt");
```

#### 7. `파일크기(경로) → 정수`
파일 크기를 바이트로 반환합니다.

```c
int64_t size = 파일크기("data.txt");
```

#### 8. `파일전체읽기(경로) → 문자열`
파일의 모든 내용을 메모리에 읽습니다.

```c
char* content = 파일전체읽기("data.txt");
```
