# 🦎 FreeLiulia 표준 라이브러리

**버전**: Phase 4
**함수 개수**: 30+
**카테고리**: 문자열 | 수학 | 배열 | 타입

---

## 📚 함수 카테고리

### 📝 문자열 함수 (10개)

#### 1. `길이(문자열) → 정수`
문자열의 길이를 반환합니다.

```freeliulia
변수 텍스트 = "안녕하세요"
변수 크기 = 길이(텍스트)  # 5
```

#### 2. `포함(문자열, 부분문자열) → 논리값`
문자열이 부분 문자열을 포함하는지 확인합니다.

```freeliulia
변수 있음 = 포함("hello world", "world")  # 참
```

#### 3. `시작(문자열, 접두어) → 논리값`
문자열이 특정 값으로 시작하는지 확인합니다.

```freeliulia
변수 시작함 = 시작("Python", "Py")  # 참
```

#### 4. `끝(문자열, 접미어) → 논리값`
문자열이 특정 값으로 끝나는지 확인합니다.

```freeliulia
변수 끝남 = 끝("hello.txt", ".txt")  # 참
```

#### 5. `찾기(문자열, 부분문자열) → 정수`
부분 문자열의 위치를 찾습니다 (없으면 -1).

```freeliulia
변수 위치 = 찾기("hello world", "world")  # 6
```

#### 6. `대문자(문자열) → 문자열`
문자열을 대문자로 변환합니다.

```freeliulia
변수 결과 = 대문자("hello")  # "HELLO"
```

#### 7. `소문자(문자열) → 문자열`
문자열을 소문자로 변환합니다.

```freeliulia
변수 결과 = 소문자("HELLO")  # "hello"
```

#### 8. `연결(문자열1, 문자열2) → 문자열`
두 문자열을 연결합니다.

```freeliulia
변수 결과 = 연결("안녕", "하세요")  # "안녕하세요"
```

#### 9. `자르기(문자열, 시작, 길이) → 문자열`
문자열의 부분을 추출합니다.

```freeliulia
변수 결과 = 자르기("hello world", 0, 5)  # "hello"
```

#### 10. `바꾸기(문자열, 기존, 새것) → 문자열`
문자열의 부분을 다른 것으로 바꿉니다.

```freeliulia
변수 결과 = 바꾸기("hello world", "world", "FreeLiulia")
```

### 🔢 수학 함수 (10개)

#### 1. `절대값(정수) → 정수`
정수의 절대값을 반환합니다.

```freeliulia
변수 결과 = 절대값(-10)  # 10
```

#### 2. `절대값_실수(실수) → 실수`
실수의 절대값을 반환합니다.

```freeliulia
변수 결과 = 절대값_실수(-3.5)  # 3.5
```

#### 3. `제곱(정수) → 정수`
정수를 제곱합니다.

```freeliulia
변수 결과 = 제곱(5)  # 25
```

#### 4. `제곱근(실수) → 실수`
실수의 제곱근을 구합니다.

```freeliulia
변수 결과 = 제곱근(16.0)  # 4.0
```

#### 5. `올림(실수) → 실수`
실수를 올림합니다.

```freeliulia
변수 결과 = 올림(3.2)  # 4.0
```

#### 6. `내림(실수) → 실수`
실수를 내림합니다.

```freeliulia
변수 결과 = 내림(3.8)  # 3.0
```

#### 7. `반올림(실수) → 실수`
실수를 반올림합니다.

```freeliulia
변수 결과 = 반올림(3.5)  # 4.0
```

#### 8. `최대값(정수, 정수) → 정수`
두 정수 중 최대값을 반환합니다.

```freeliulia
변수 결과 = 최대값(5, 10)  # 10
```

#### 9. `최소값(정수, 정수) → 정수`
두 정수 중 최소값을 반환합니다.

```freeliulia
변수 결과 = 최소값(5, 10)  # 5
```

#### 10. `거듭제곱(실수, 정수) → 실수`
밑을 지수로 거듭제곱합니다.

```freeliulia
변수 결과 = 거듭제곱(2.0, 3)  # 8.0
```

### 📊 배열 함수 (8개)

#### 1. `배열길이(배열) → 정수`
배열의 길이를 반환합니다.

```freeliulia
변수 배열 = [1, 2, 3, 4, 5]
변수 크기 = 배열길이(배열)  # 5
```

#### 2. `배열합계(배열) → 정수`
배열의 모든 요소의 합을 반환합니다.

```freeliulia
변수 배열 = [1, 2, 3, 4, 5]
변수 합 = 배열합계(배열)  # 15
```

#### 3. `배열평균(배열) → 실수`
배열의 평균을 반환합니다.

```freeliulia
변수 배열 = [1, 2, 3, 4, 5]
변수 평균 = 배열평균(배열)  # 3.0
```

#### 4. `배열최대(배열) → 정수`
배열의 최대값을 반환합니다.

```freeliulia
변수 배열 = [1, 5, 3, 9, 2]
변수 최대 = 배열최대(배열)  # 9
```

#### 5. `배열최소(배열) → 정수`
배열의 최소값을 반환합니다.

```freeliulia
변수 배열 = [1, 5, 3, 9, 2]
변수 최소 = 배열최소(배열)  # 1
```

#### 6. `배열포함(배열, 값) → 논리값`
배열에 특정 값이 포함되어 있는지 확인합니다.

```freeliulia
변수 배열 = [1, 2, 3, 4, 5]
변수 있음 = 배열포함(배열, 3)  # 참
```

## 📖 사용 예제

### 예제 1: 텍스트 처리

```freeliulia
변수 입력 = "hello world"
변수 크기 = 길이(입력)

변수 위치 = 찾기(입력, "world")

만약 포함(입력, "world")
    변수 결과 = "world를 찾았습니다"
끝
```

### 예제 2: 수학 계산

```freeliulia
변수 x = -5
변수 abs_x = 절대값(x)

변수 제곱값 = 제곱(4)
변수 근값 = 제곱근(16.0)

변수 최대 = 최대값(10, 20)
변수 최소 = 최소값(10, 20)
```

### 예제 3: 배열 분석

```freeliulia
변수 점수 = [85, 90, 78, 92, 88]

변수 합계 = 배열합계(점수)

변수 최저 = 배열최소(점수)

만약 배열포함(점수, 90)
```
