## 🎨 FreeLiulia 언어 특징

### 1️⃣ 한글 문법
```
함수 더하기(가: 정수, 나: 정수) -> 정수
    반환 가 + 나
끝

변수 결과: 정수 = 더하기(3, 4)
출력(결과)  # 7
```

### 2️⃣ 다중 디스패치
```
함수 처리(x: 정수) -> 문자열
    반환 "정수"
끝

함수 처리(x: 실수) -> 문자열
    반환 "실수"
끝

출력(처리(42))      # "정수"
출력(처리(3.14))    # "실수"
```

### 3️⃣ 고성능 (JIT 컴파일)
- 제1차 호출: 컴파일 (타입 특화)
- 이후 호출: 네이티브 속도

### 4️⃣ 벡터화
```
배열 arr = [1, 2, 3, 4, 5]
배열 결과 = arr .* 2  # 요소별 곱셈
# 결과 = [2, 4, 6, 8, 10]
```

## 📝 코드 예제

### 예제 1: 소수 찾기
```
함수 소수인가(n: 정수) -> 논리값
    만약 n < 2
        반환 거짓
    끝

    반복 i = 2 부터 n 까지
        만약 n % i == 0
            반환 거짓
        끝
    끝

    반환 참
끝

반복 num = 1 부터 20 까지
    만약 소수인가(num)
        출력(num)
    끝
끝
```

### 예제 2: 벡터 내적
```
함수 내적(v1: 배열<실수>, v2: 배열<실수>) -> 실수
    변수 합계: 실수 = 0.0
    반복 i = 1 부터 길이(v1) 까지
        합계 = 합계 + (v1[i] * v2[i])
    끝
    반환 합계
끝

배열 벡터1 = [1.0, 2.0, 3.0]
배열 벡터2 = [4.0, 5.0, 6.0]
출력(내적(벡터1, 벡터2))  # 32.0
```
