## 📋 구현된 함수 목록

### 📝 문자열 함수

1. **길이(문자열) → 정수** - 문자열의 길이 반환
2. **포함(문자열, 부분) → 논리값** - 부분 문자열 포함 여부
3. **시작(문자열, 접두어) → 논리값** - 시작 문자열 확인
4. **끝(문자열, 접미어) → 논리값** - 끝 문자열 확인
5. **찾기(문자열, 부분) → 정수** - 부분 문자열 위치 (없으면 -1)
6. **대문자(문자열) → 문자열** - 대문자로 변환
7. **소문자(문자열) → 문자열** - 소문자로 변환
8. **연결(문자열1, 문자열2) → 문자열** - 문자열 연결
9. **자르기(문자열, 시작, 길이) → 문자열** - 부분 추출
10. **바꾸기(문자열, 기존, 새것) → 문자열** - 부분 교체

### 🔢 수학 함수

1. **절대값(정수) → 정수** - 정수의 절대값
2. **절대값_실수(실수) → 실수** - 실수의 절대값
3. **제곱(정수) → 정수** - 제곱
4. **제곱근(실수) → 실수** - 제곱근
5. **올림(실수) → 실수** - 올림
6. **내림(실수) → 실수** - 내림
7. **반올림(실수) → 실수** - 반올림
8. **최대값(정수, 정수) → 정수** - 최대값
9. **최소값(정수, 정수) → 정수** - 최소값
10. **거듭제곱(실수, 정수) → 실수** - 거듭제곱

### 📊 배열 함수

1. **배열길이(배열) → 정수** - 배열 길이
2. **배열합계(배열) → 정수** - 모든 요소의 합
3. **배열평균(배열) → 실수** - 평균값
4. **배열최대(배열) → 정수** - 최대값
5. **배열최소(배열) → 정수** - 최소값
6. **배열포함(배열, 값) → 논리값** - 값 포함 여부

### 🔍 타입 검사 함수

1. **정수인가(값) → 논리값** - 정수 타입 확인
2. **실수인가(값) → 논리값** - 실수 타입 확인
3. **논리값인가(값) → 논리값** - 논리값 타입 확인
4. **문자열인가(값) → 논리값** - 문자열 타입 확인
